WHAT IS IT?

Asgard is a small independent file system library designed for smart 
card operating system. it support standard ISO7816-4/GSM 11.11 commands 
such as select, read binary, read record, write binary, write record.
it provide high level access to read/write from any memory using tree
alike file structure.
